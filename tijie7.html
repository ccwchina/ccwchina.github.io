题解：AT_joi2024_yo1c_b 桁 (Digit)

思路

题目要我们求 A 与 B 的和在十进制下的位数，我们需要先求出 A 与 B 的和，然后对和不断进行除以 10 的操作，这样可以删掉和的最后一位，举个例子： 假设 A 与 B 的和是 123， 123 除以 10 会得到 12， 12 再除以 10 会得到 1， 1 再除以 10 会得到 0， 此时跳出循环。 再设置一个计数器，循环过程中每次除以 10 的操作进行完毕后将计数器增加 1，最后输出即可。

代码

#include<iostream> using namespace std; int main() { int a,b; cin>>a>>b; a+=b; int ans=0; while(a!=0) { a/=10; ans++; } cout<<ans; return 0; }
