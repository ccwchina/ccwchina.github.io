题解：AT_joi2024_yo1c_c 揃った文字 (Matched Letters)

思路

读入字符串，然后循环判断是否符合条件，如果不符合条件直接输出No，然后结束程序，否则进行到最后，输出Yes。

代码

#include<bits/stdc++.h> using namespace std; int main() { int n; string s; cin>>n>>s; for(int i=0;i<n-1;i++) { if(s[i]!=s[i+1]) { cout<<"No"; return 0; } } cout<<"Yes"; return 0; }
