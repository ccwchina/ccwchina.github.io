题解：AT_joi2023_yo1b_c 繰り返し文字列 (Repeating String)

思路

这道题可以枚举，从中间分割，从第 1 个字符和第 N/2+1 开始比较，比较到第 N/2 个字符和第 N个字符，比较的过程中如果发现有不同的直接输出 No，结束程序，否则执行到最后输出 Yes。

代码

#include<bits/stdc++.h> using namespace std; int main() { int n; cin>>n; string s; cin>>s; for(int i=0;i<n/2;i++) { if(s[i]!=s[i+n/2]) { cout<<"No"; return 0; } } cout<<"Yes"; return 0; }
